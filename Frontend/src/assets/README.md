# Logo Instructions

## Adding Your NorthSoul Clothing Logo

1. Save your logo image as `northsoulclothing.png` in this directory
2. Recommended dimensions: 32x32px or 64x64px for best quality
3. Supported formats: PNG (recommended), JPG, SVG

## Current Logo Path
The navbar is configured to look for: `/src/assets/northsoulclothing.png`

## Fallback
If the logo image is not found, the text "NorthSoul Clothing" will be displayed instead.
